Módulo 1: Fundamentos de Programación en Java

1. Variables y Tipos de Datos

En Java existen tipos primitivos y sus equivalentes como clases Wrapper para manejar datos de manera más flexible.

Tipos Primitivos

  • int → números enteros (ej: 5, 100, -25)
  • double → números decimales (ej: 3.14, 2.5, -0.75)
  • char → un solo carácter (ej: 'A', '5', '@')
  • boolean → valores lógicos true o false
  • byte, short, long, float → otros tipos numéricos
int edad = 25; double precio = 19.99; char inicial = 'J'; boolean activo = true;

Clases Wrapper

Cada tipo primitivo tiene su clase equivalente:

  • int → Integer
  • double → Double
  • char → Character
  • boolean → Boolean
Ventajas de las Wrapper:
• Autoboxing/Unboxing automático
• Métodos útiles como Integer.parseInt("123")
• Constantes como Integer.MAX_VALUE
Integer numero = 42; // Autoboxing String texto = "123"; int convertido = Integer.parseInt(texto); int maximo = Integer.MAX_VALUE;

2. Operadores Aritméticos

Los operadores aritméticos nos permiten realizar cálculos matemáticos básicos:

  • + → suma
  • - → resta
  • * → multiplicación
  • / → división
  • % → módulo (residuo de la división)
int a = 10; int b = 3; System.out.println(a + b); // 13 System.out.println(a - b); // 7 System.out.println(a * b); // 30 System.out.println(a / b); // 3 System.out.println(a % b); // 1 (residuo)
Tip: El operador módulo (%) es muy útil para determinar si un número es par o impar: si numero % 2 == 0, entonces es par.

3. Condicionales

Las estructuras condicionales nos permiten ejecutar código según ciertas condiciones.

if - else if - else

Ejecuta bloques de código dependiendo de condiciones booleanas:

int edad = 18; if (edad >= 18) { System.out.println("Eres mayor de edad"); } else if (edad >= 13) { System.out.println("Eres adolescente"); } else { System.out.println("Eres menor de edad"); }

switch

Evalúa una variable contra múltiples opciones específicas:

int dia = 1; switch (dia) { case 1: System.out.println("Lunes"); break; case 2: System.out.println("Martes"); break; default: System.out.println("Otro día"); break; }
Operadores de comparación:
== (igual), != (diferente), < (menor), > (mayor), <= (menor igual), >= (mayor igual)

4. Bucles (Ciclos)

Los bucles nos permiten repetir código múltiples veces de manera eficiente.

for

Se usa cuando sabes exactamente cuántas veces repetir algo:

// Imprime números del 1 al 5 for (int i = 1; i <= 5; i++) { System.out.println("Número: " + i); } // Recorrer un arreglo int[] numeros = {1, 2, 3, 4, 5}; for (int numero : numeros) { System.out.println(numero); }

while

Repite mientras una condición sea verdadera:

int contador = 0; while (contador < 3) { System.out.println("Iteración: " + contador); contador++; }

do-while

Siempre se ejecuta al menos una vez:

int numero; do { numero = (int)(Math.random() * 10) + 1; System.out.println("Número generado: " + numero); } while (numero != 5);

5. Sentencias de Control de Flujo

Estas sentencias nos permiten controlar el flujo de ejecución en bucles y métodos.

break

Termina completamente el bucle actual:

for (int i = 1; i <= 10; i++) { if (i == 5) { break; // Sale del bucle cuando i es 5 } System.out.println(i); // Imprime 1, 2, 3, 4 }

continue

Salta a la siguiente iteración del bucle:

for (int i = 1; i <= 5; i++) { if (i == 3) { continue; // Salta cuando i es 3 } System.out.println(i); // Imprime 1, 2, 4, 5 }

return

Termina un método y opcionalmente devuelve un valor:

public static int suma(int a, int b) { return a + b; // Termina el método y devuelve la suma } public static void saludar() { System.out.println("Hola!"); return; // Opcional en métodos void }
Tip práctico: Usa break para salir de bucles cuando encuentres lo que buscas, y continue para saltarte elementos que no necesitas procesar.
